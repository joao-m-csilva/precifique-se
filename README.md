# PrecifiqueSE

O **PrecifiqueSE** foi desenvolvido para ajudar profissionais a calcularem um valor mais justo para seus serviços.

A ideia do projeto é partir do salário mensal desejado e transformar esse valor em uma referência de cobrança por hora, por dia e por projeto. Dessa forma, fica mais simples estimar propostas considerando tempo de trabalho e custos extras, como assinaturas de IA, ferramentas ou outros recursos necessários.

## Stack utilizada

- React
- Vite
- JavaScript
- CSS
- Lucide React
